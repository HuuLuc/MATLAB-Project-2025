Giới thiệu

Đây là đồ án xây dựng ứng dụng MATLAB (App Designer) hỗ trợ giải các bài toán phương pháp số thường gặp. Ứng dụng được thiết kế dạng nhiều tab, giúp người dùng nhập dữ liệu, chọn phương pháp và quan sát kết quả trực quan.

Chức năng chính
- Tìm nghiệm phương trình: Chia đôi, Lặp đơn, Newton
- Nội suy & Hồi quy: Lagrange, Newton, Hồi quy tuyến tính và phi tuyến
- Tính gần đúng đạo hàm: Sai phân tiến, lùi, trung tâm
- Tính gần đúng tích phân: Hình thang, Simpson 1/3, Simpson 3/8

Công cụ sử dụng: MATLAB App Designer, Github (quản lý code và cộng tác nhóm),...
